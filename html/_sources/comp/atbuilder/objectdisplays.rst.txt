.. _objectdisplays:

Object Displays
###############

This chapter contains the description of all predesigned object displays which can be used for visualization. The appearance and behavior of these object displays can be customized by parameters. atvise provides object displays in different styles (Advanced, Standard, Steelblue). If a parameter is exclusive for a specific style, it is indicated by :stdstyle:`Standard` for the styles *Standard* and *Steelblue* or :advstyle:`Advanced` for the style *Advanced*.
