Configuration - Series
######################

This display provides the management of the data sets for the chart.

.. rubric:: Parameters

This control has the following parameters:

- :advstyle:`Advanced` **font color** -- Color which will be applied to the labels (default: global parameter atvFontColor)
- :advstyle:`Advanced` **fill color** -- Left sidebar fill color (default: global parameter atvFillColor)
- **node entry path** -- Defines the entry path in the configuration dialog (default: AGENT.OBJECTS).

Subdisplay:

- **display for message** -- Display for showing messages (default: "Message Dialog" object display)
- **display for series bar** -- Display for showing a bar series (default: "Configuration - Bar/Column" object display)
- **display for series pie** -- Display for showing a pie series (default: "Configuration - Pie" object display)
- **display for series line** -- Display for showing a line series (default: "Configuration - Line/Area" object display)

Appearance: 

- **button color 1** -- Color for *lower* and *upper value address* button when clicked (default: #c9d9b8 (advanced), #595959 (standard))
- **button color 2** -- Color for *lower* and *upper value address* button (default: #d7d7d7 (advanced), #000000 (standard))
- :stdstyle:`Standard` **number of series** -- The number of displayed series (default: 10)
