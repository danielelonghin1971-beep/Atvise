Clock Combined
##############

This control displays the current client time and date.

.. rubric:: Parameters

- :advstyle:`Advanced` **font color** -- Color which will be applied to the clock's labeling (default: global parameter atvFontColor)
- :advstyle:`Advanced` **fill color** -- Fill color of the clock frame (default: global parameter atvFillColor)
- :advstyle:`Advanced` **border color** -- Color of the button's border (default: global parameter atvBorderColor)
- :advstyle:`Advanced` **seconds-pointer color** -- Color of the clock's second-pointer (default: global parameter atvAccent1)
- :advstyle:`Advanced` **symbol color** -- Color of the clock's scale symbols (default: global parameter atvSymbolColor)
- **visibility date** -- Turns the display of date on/off (default: visible)
- **visibility digital time** -- Turns the display of digital time on/off (default: hidden)
