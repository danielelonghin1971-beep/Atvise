Button Table Notification
#########################

This display supplements the table with a notification button for calling additional information in e.g. Fault. A special configuration, as for the table, is not required for this display. All you have to do is set the table name and font in the parameters.

.. rubric:: Conditions

This display can only be used together with a table!

.. rubric:: Parameters

- **table ID** -- Unique table name (default: atvise_table_1)

Appearance:

- **font family** -- Font family (default: Arial)
- **font size** -- Font sizes (default: 12)
- **font color** -- Font color and icon color (default: SYSTEM.GLOBALS.atvSymbolColor)

Options:

- **message board** --  shows a dialog for messages, information or warnings (default: message board OD)
