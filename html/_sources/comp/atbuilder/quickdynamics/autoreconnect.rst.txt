.. _autoreconnect:

Auto Reconnect
##############

This Quick Dynamic enables the auto reconnect functionality and provides an additional keepalive mechanism.

.. attention:: The auto reconnect functionality is activated by default with a 5 seconds interval; the keepalive mechanism is activated too. Add the Quick Dynamic to the Default Display and adjust the parameters to change the default behavior.

.. rubric:: Parameters:

- **activated** (*activates auto reconnect*) -- Use this parameter for enabling or disabling the auto reconnect functionality. (default: true)
- **interval** (*interval in seconds for auto reconnect*) -- This parameter defines the time span for auto reconnect in seconds. (default: 5)
- **defaultconfiguration** (hidden) -- Defines if the passed parameters shall be set as default. Internally used by atvise.

.. hint:: The parameter for the interval in seconds for verifying if server is alive (keepalive-mechanism) is omitted in this Quick Dynamic. The keepalive-interval can be configured in the :ref:`webmicfg.js <visu_configuration>`.
